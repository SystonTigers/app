| `POST /push/register`  `POST /push/send`        | Notifications                        |
| `GET /health`                                   | Heartbeat + fallback switch          |

# 🔁 6.  Automation Blueprints (Starter via Make)

**Scenario 1 – Live Match Webhook**
Trigger on Sheet edit → POST `/events/live` → GitHub/R2 write → mark row Posted.

**Scenario 2 – League Table Sync**
Every 6 h → pull table → hash compare → if changed, update R2 file + tweet.

**Scenario 3 – Daily Posts**
06:00 → fetch birthdays/quotes → compose payload → POST `/render` → auto-post.

**Scenario 4 – Fixture Countdown**
08:00 → filter fixtures within 24 h → POST `/render`.

**Scenario 5 – Throwback Thursday**
Thu 19:00 → pick old photo → POST `/render`.

**Scenario 6 – Usage Counter**
First module = `/usage/make/increment`; run stops if over limit.

**Total ≈ 250 ops/month**.
