# Brand Assets Folder

This folder contains all brand assets for video overlays and graphics.

## Folder Structure

### `/badges`
Team logos and competition badges
- Format: PNG with transparency
- Size: 512x512px recommended
- Naming: `{team-name}.png` or `home.png`, `away.png`

### `/fonts`
Custom fonts for overlays
- Format: TTF or OTF
- Include: Bold, Regular weights
- Naming: `Font-Bold.ttf`, `Font-Regular.ttf`

### `/luts`
Color grading LUT files
- Format: .cube files
- Naming: `broadcast.cube`, `cinematic.cube`

### `/overlays`
Scorebug and persistent graphics
- Format: PNG with transparency
- Size: 1920x1080px
- Naming: `scorebug.png`, `lower-third.png`

### `/slates`
Opening and closing slates
- Format: PNG or MP4
- Size: 1920x1080px
- Naming: `opening-slate.png`, `closing-slate.png`

### `/stingers`
Transition effects and stingers
- Format: MP4 with alpha channel
- Duration: 0.5-2 seconds
- Naming: `transition-in.mp4`, `transition-out.mp4`
